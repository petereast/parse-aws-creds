# parse-aws-creds

A tiny utility that runs converts AWS sts JSON responses into credentials file structure.

## Usage

```
parse-aws-creds 0.1.0

USAGE:
    parse-aws-creds [FLAGS]

FLAGS:
    -d, --default-header    
    -h, --help              Prints help information
    -V, --version           Prints version information
    -w, --write             
```
